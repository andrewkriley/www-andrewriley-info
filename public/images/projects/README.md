# Project Images

Use this folder for technology, AI tooling, homelab, automation, and portfolio case-study imagery.

Recommended sizes:

- Project card images: `1600 x 1000px`.
- Case-study header images: `1920 x 1080px`.
- Inline diagrams/screenshots: at least `1400px` wide.

Suggested naming:

- `ai-tooling-cover.webp`
- `homeaiops-cover.webp`
- `splunk-mcp-cover.webp`
- `portfolio-build-cover.webp`

Prefer images that show the work clearly: dashboards, architecture sketches, terminal output, hardware, workflows, or before/after states.

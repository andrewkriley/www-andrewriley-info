# Music Images

Use this folder for DJ Riles, decks, events, venue shots, cover art, and music-related creative imagery.

Recommended sizes:

- Music card images: `1600 x 1200px`, 4:3.
- Wide section images: `1920 x 1080px`, 16:9.
- Cover-art style images: `1200 x 1200px`, square.

Suggested naming:

- `dj-riles-cover.webp`
- `decks-detail.webp`
- `music-portrait.webp`
- `mix-cover-YYYY-MM.webp`

Look for images with energy, warmth, movement, light, and texture.

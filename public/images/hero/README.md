# Hero Image Placeholders

Replace these files with final imagery when available:

- `professional-placeholder.svg` - professional portrait, speaking, work, or leadership image.
- `diy-placeholder.svg` - workshop, tools, renovation, or hands-on project image.
- `music-placeholder.svg` - DJ, decks, venue, records, or music-related image.

Recommended replacement size: `1600 x 1200px` at a `4:3` aspect ratio.

Use the same filenames to update the landing page without code changes, or update `heroImages` in `src/data/site.ts` if the filenames change.

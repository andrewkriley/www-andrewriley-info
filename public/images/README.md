# Image Inventory

This directory is the source for site images that are served from `/images/...`.

Keep final images reasonably compressed and prefer these formats:

- `.webp` for photos and rich images.
- `.png` only when transparency is needed.
- `.svg` for simple vector artwork, logos, and placeholders.

Suggested top-level groups:

- `hero/` - landing page hero images.
- `profile/` - portraits and biography images.
- `projects/` - technology, AI, homelab, and case-study imagery.
- `diy/` - workshop, tools, and physical build photos.
- `music/` - DJ Riles, decks, venues, cover art, and music-related photos.
- `play/` - images used by the Play page and DJ Riles hero sections.
- `writing/` - post cover images and article illustrations.
- `social/` - Open Graph and share-card artwork.

# Social Images

Use this folder for Open Graph, share cards, and social preview images.

Recommended sizes:

- Default Open Graph image: `1200 x 630px`.
- Page-specific Open Graph images: `1200 x 630px`.
- Square avatar/share image: `1200 x 1200px`.

Suggested naming:

- `og-default.png`
- `og-build.png`
- `og-make.png`
- `og-create.png`
- `og-writing.png`

Keep text large, high contrast, and readable when the image is shown small in link previews.

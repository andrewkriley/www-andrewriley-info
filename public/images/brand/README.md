# Brand Images

Use this folder for reusable brand assets and visual identity files.

Recommended assets:

- `mark.svg` - simple personal/site mark if one is created.
- `logo.svg` - full wordmark if needed.
- `favicon-source.svg` - editable favicon source.
- `texture.webp` - optional background texture or visual motif.

Keep brand assets source-like and reusable. Export final favicon files separately if the app starts referencing them directly.

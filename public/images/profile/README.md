# Profile Images

Use this folder for portraits, speaking photos, and biography images.

Recommended sizes:

- `profile-primary.webp` - `1200 x 1200px`, square portrait.
- `profile-wide.webp` - `1600 x 900px`, wide crop for about/contact sections.
- `profile-candid.webp` - `1600 x 1200px`, natural or behind-the-scenes image.

Aim for warm, human images that still feel professional.

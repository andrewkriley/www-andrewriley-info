# DIY Images

Use this folder for hands-on making, workshop, renovation, tools, and physical build photos.

Recommended sizes:

- Build card images: `1600 x 1200px`, 4:3.
- Process photos: `1600 x 1200px`, 4:3.
- Wide project headers: `1920 x 1080px`, 16:9.

Suggested naming:

- `workshop-cover.webp`
- `project-name-before.webp`
- `project-name-process-01.webp`
- `project-name-after.webp`

Useful sets include before, during, after, details, mistakes, and finished result.
